CSS / LESS

Our CSS is built on Less, a preprocessor with additional functionality like variables, mixins, and functions for compiling CSS.

Variables

Variables are used throughout the entire project as a way to centralize and share commonly used values like colors, spacing, or font stacks.

Colors

Easily make use of of the SpotXchange colors by using these variables.

@spotx-green:        #8ec641;
@spotx-dk-green:     #74a636;
@spotx-blue:         #1b9dd0;
@spotx-dk-blue:      #1f4389;
@spotx-grey:         #bcbec0;
@spotx-dk-grey:      #808285;

Colors

Our gradients use a mixin to simplify the css. Add the gradient-green-horizontal class to any element for the green horizontal gradient.

.gradient-green-horizontal {  
  #gradient > .horizontal(#9ac83c, #75a637);  
}
.gradient-green-vertical {  
  #gradient > .vertical(#9ac83c, #75a637);  
}
.gradient-blue-horizontal {  
  #gradient > .horizontal(#1b9dd0, #0080b5);  
}
.gradient-blue-vertical {  
  #gradient > .vertical(#1b9dd0, #0080b5);  
}
.gradient-grey-horizontal {  
  #gradient > .horizontal(#fff, #e0e0e0);  
}
.gradient-grey-vertical {  
  #gradient > .vertical(#fff, #e0e0e0);  
}
