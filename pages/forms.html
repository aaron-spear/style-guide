Tooltips

Tooltips need to be applied to the inputs and <span> tags around the label. Use tooltip-placement="right" and tooltip-trigger="mouseenter" to determine placement and functionality.

Name is required 
<div class="form-group required">
  <label for="name" class="col-sm-3 control-label">
    <span tooltip="This name will be used to identify the campaign in reports and the UI." tooltip-trigger="mouseenter">Name:</span>
  </label>
  <div class="col-sm-5 error-container">
    <span class="help-block">Name is required</span>
    <input type="text" class="form-control" tooltip="Short name to identify this campaign" tooltip-trigger="mouseenter" id="name">				      
  </div>
</div>
<div class="form-group">
  <label for="impressions" class="col-sm-3 control-label">
    <span tooltip="Enter the number of impressions from the insertion order for this campaign to help you monitor delivery." tooltip-trigger="mouseenter">Impressions Ordered:</span>
  </label>
  <div class="col-sm-5">
    <input type="text" class="form-control input-sm-2" tooltip="Total impressions for this campaign" tooltip-placement="right" tooltip-trigger="mouseenter" id="impressions">
  </div>
</div>

Sizes

Create taller or shorter form controls that match button sizes by using .input-lg or .input-sm classes.

<input class="form-control input-lg" type="text" placeholder=".input-lg">
<input class="form-control" type="text" placeholder="Default input">
<input class="form-control input-sm" type="text" placeholder=".input-sm">

<select class="form-control input-lg">...</select>
<select class="form-control">...</select>
